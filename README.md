# TateJakeWpfApp
画像やテキスト合成のサンプルアプリケーションです。
イメージ画像（ジャケット画像など）を上部に配置することを前提に、タイトルなどを下部に配置し、プレビューで位置を調整した上で、選択した画像（複数可）とタイトルを合成し、縦長のジャケット画像を生成します。<br/>
