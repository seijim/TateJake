# TateJakeWpfApp
画像やテキスト合成のサンプルアプリケーションです。
イメージ画像（ジャケット画像など）を上部に配置し、タイトルなどを下部に配置して、プレビューで位置を調整した上で、選択した画像（複数可）とタイトルを合成し、縦長のジャケット画像を生成します。<br/><br/>
シンプルな編集画面です。「ファイル」メニューから設定画面を表示すると、初期設定値を編集することもできます。<br/>
<img src="./images/01.png" width="80%" /><br/><br/>
タイトルを入力して、「テキスト プレビュー」ボタンを押して、表示を確認します。<br/>
<img src="./images/02.png" width="80%" /><br/><br/>
テキストボックス内のテキストの横には、画面上に表示されているフォントサイズ、Ｘ座標、Ｙ座標が自動的に付与されます。<br/>
<img src="./images/03.png" width="80%" /><br/><br/>
３行目のテキストがやや左に寄っている為、Ｘ座標を修正した後、「テキスト プレビュー」ボタンを押して、表示を確認します。問題なければ、「画像選択 ＆ 合成」ボタンを押します。
<img src="./images/04.png" width="80%" /><br/><br/>
画像ファイルを１つ、または、複数選択し、「開く」を押すと、画像ファイルのあるフォルダーに「output」というフォルダーが作成され、その中に合成された画像ファイルが出力されます。
<img src="./images/05.png" width="50%" /><br/><br/>
合成された画像ファイルを確認します。<br/>
